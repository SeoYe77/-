# 基于城市场景仿真序列的低空多目标跟踪滤波方法研究

## 摘要
面向城市场景下6—8架低空无人机目标在尺度变化、检测噪声、短时漏检和遮挡条件下容易发生轨迹中断与ID切换的问题，本文提出一种基于城市场景仿真序列的低空多目标跟踪滤波方法。该方法由检测序列预处理、卡尔曼滤波状态预测和数据关联更新三部分组成：首先对检测结果进行置信度筛选与尺度归一化，以降低误检和检测抖动对关联过程的影响；随后利用卡尔曼滤波对目标位置和尺度进行时序预测，以补偿短时漏检并缓解透视投影引起的尺度波动；最后结合匈牙利算法、IOU和马氏距离完成目标匹配，并通过轨迹确认与保留机制降低误匹配与ID切换。实验结果表明，该方法能够有效提升城市场景仿真序列中的多目标跟踪连续性，减少ID切换，提高低空多目标跟踪滤波的稳定性和鲁棒性。

**关键词：** 城市场景仿真；低空无人机；多目标跟踪；卡尔曼滤波；ID切换

## 1 引言
随着低空领域开放与无人机技术的快速发展，低空无人机目标在城市空域中的监测、告警与处置需求日益增长。由于城市建筑遮挡、目标尺度随距离快速变化、检测噪声和短时漏检等因素的共同影响，低空多目标跟踪过程中容易出现轨迹碎片化、跟踪中断和ID切换频繁等问题，难以满足稳定、连续的跟踪需求。

目标检测与多目标跟踪是低空目标感知的关键环节，其中检测结果的稳定性和数据关联的准确性直接决定后续跟踪性能。近年来，YOLO系列检测器因其实时性和检测精度被广泛用于无人机目标检测；DeepSORT、ByteTrack、BoT-SORT等多目标跟踪方法通过运动预测、数据关联和轨迹管理提升了复杂场景下的跟踪鲁棒性。尤其是卡尔曼滤波在状态估计与短时预测方面具有良好效果，能够有效缓解遮挡、漏检和观测噪声带来的影响。

然而，真实低空无人机视频数据获取困难、场景可控性差、标注成本高，难以支持可重复、可对比的算法验证。因此，本文基于城市场景仿真序列构建低空多目标跟踪实验条件，在6—8架无人机的仿真检测序列上，对不同关联方式与运动预测方法进行分析，重点研究卡尔曼滤波在降低ID切换、提升轨迹连续性方面的作用，并进一步提出一种适用于城市场景仿真序列的低空多目标跟踪滤波方法。

本文的主要工作与贡献如下：
1. 基于城市场景仿真序列构建低空多目标跟踪任务，统一考虑透视投影、检测噪声、漏检和误检等因素；
2. 提出融合检测预处理、卡尔曼滤波预测和多度量数据关联的低空多目标跟踪滤波方法；
3. 针对ID切换问题，引入轨迹确认与保留机制，增强复杂城市场景下的轨迹稳定性；
4. 通过仿真实验验证所提方法在减少ID切换和提升跟踪连续性方面的有效性。

## 2 系统总体描述与问题定义
本文研究的对象为城市场景下的低空多目标跟踪任务，重点分析噪声、漏检、尺度变化和遮挡条件下的关联稳定性问题。

设场景内共有N架低空无人机目标，在时刻t，第k个目标的真实状态表示为：

$$X_k(t) = [x(t), y(t), z(t), \dot{x}(t), \dot{y}(t), \dot{z}(t)]^T$$

其中，$(x,y,z)$表示三维空间位置，$(\dot{x},\dot{y},\dot{z})$表示速度。

通过虚拟相机的透视投影，三维目标投影到图像平面后得到二维检测框：

$$b_k(t) = [u(t), v(t), w(t), h(t)]^T$$

其中，$(u,v)$为检测框中心坐标，$w,h$为宽和高。

在实际观测中，检测结果受到噪声、误检和漏检影响，可表示为：

$$z_k(t) = b_k(t) + n_k(t)$$

其中，$z_k(t)$为检测器输出，$n_k(t) \sim N(0,R)$为高斯观测噪声。

多目标跟踪的目标是从含噪、不完整的检测序列中恢复连续轨迹，并为每个目标分配稳定ID。为此，本文采用以检测结果为输入、以运动预测和数据关联为核心的低空多目标跟踪滤波方法，对城市场景仿真序列中的轨迹连续性和ID切换问题进行分析与优化。

## 3 城市场景多无人机仿真与检测序列生成
### 3.1 三维场景与目标运动模型
仿真场景定义为城市低空三维空间：

$$\Omega = [X_{\min}, X_{\max}] \times [Y_{\min}, Y_{\max}] \times [Z_{\min}, Z_{\max}]$$

本文默认设置场景内无人机目标数量为7架，以覆盖城市低空小规模多目标跟踪的典型情形。

城市建筑群采用随机生成方式构建，以模拟真实楼宇遮挡环境。禁飞区采用球体模型表示，并通过吸引行为与边界反弹机制控制目标运动，使目标在仿真空间内形成更接近真实低空环境的运动轨迹。

### 3.2 虚拟相机模型与透视投影
虚拟相机采用针孔成像模型，三维空间点投影到二维图像平面后，目标尺度会随深度变化而改变。由于目标距离相机越远，检测框越小，因此同一目标在相邻帧之间可能出现显著尺度波动。这一现象会降低IOU匹配的稳定性，是造成ID切换的重要原因之一。

### 3.3 检测噪声与漏检/误检建模
为模拟真实检测器输出，本文对检测结果引入高斯位置噪声、高斯尺度噪声、伯努利漏检和泊松误检模型。由此生成的检测序列包含真实目标检测框与误检框，并输出为标准化格式，能够直接作为多目标跟踪算法输入。

上述仿真设置使检测序列具有较强挑战性，尤其是在尺度快速变化、短时漏检和误检干扰条件下，对后续数据关联与轨迹管理提出了更高要求。

## 4 低空多目标跟踪滤波方法
本文提出的低空多目标跟踪滤波方法主要由三个部分组成：检测序列预处理、运动状态估计以及数据关联与轨迹维护。该方法在保留传统多目标跟踪数学原理的基础上，针对城市场景仿真序列中的目标尺度变化、检测噪声、短时漏检和ID切换问题进行了面向性的改进设计。

### 4.1 多目标跟踪问题形式化定义
多目标跟踪（Multi-Object Tracking, MOT）的核心任务是：从逐帧检测框序列中，为每个目标分配唯一且稳定的身份标识（ID），并输出连续的时空轨迹。

设仿真序列总帧数为T，第t帧（t=1,2,...,T）的检测结果为$D_t = \{z_i^t\}_{i=1}^{M_t}$，其中$M_t$为第t帧检测框数量，单个检测框$z_i^t = [x_i^t, y_i^t, w_i^t, h_i^t]^T$表示第i个目标的检测框位置与尺度信息。

多目标跟踪的输出为轨迹集合
$\mathcal{T} = \{\tau_j\}_{j=1}^{N_{track}}$。
其中单条轨迹表示为：

$$\tau_j = \{(z_{j,t}, t) \mid t \in [t_{j,start}, t_{j,end}]\}$$

$N_{track}$为生成的独立轨迹 ID 总数。跟踪的核心目标是：在检测噪声、尺度变化、随机漏检等干扰下，实现检测框与轨迹的最优数据关联，最小化轨迹碎片化与 ID 切换，最大化轨迹连续性与 ID 保持能力。

### 4.2 检测序列预处理
在复杂城市场景中，检测器输出往往伴随误检、检测抖动以及尺度波动。若直接将原始检测结果输入关联模块，容易引发匹配歧义，进而造成轨迹碎片化。因此，本文首先对检测序列进行预处理。

#### 4.2.1 YOLOv8 检测结果筛选
对 YOLOv8 输出的检测框进行初步筛选，仅保留置信度高于阈值的检测结果，剔除明显的低置信度误检框。该步骤的作用是减少后续关联过程中的错误输入，提高检测序列的可靠性。

#### 4.2.2 置信度过滤
对保留下来的检测框进一步执行置信度过滤，优先保留置信度较高的目标框。由于误检目标往往在置信度上明显低于真实目标，因此该步骤能够有效降低误检框参与关联的概率，减少由误检引起的 ID 分裂和轨迹重生。

#### 4.2.3 尺度归一化
针对城市场景中目标尺度随距离快速变化的问题，对检测框中心位置和尺度信息进行归一化处理，使不同深度下的检测结果在数值上具有更好的可比性。该步骤能够削弱透视投影带来的尺度波动对后续关联计算的影响，提高帧间匹配稳定性。

通过上述预处理，本文将原始检测结果转换为更稳定、更适合关联计算的输入序列，为后续运动状态估计和数据关联提供基础支撑。

### 4.3 运动状态估计
在低空多目标跟踪场景中，目标不仅会发生平面位移，还会因远近变化导致尺度波动。若仅依赖检测结果进行帧间匹配，短时漏检和尺度变化将直接导致轨迹中断。为此，本文引入卡尔曼滤波对目标运动状态进行估计与预测。

卡尔曼滤波的作用在于：基于上一时刻的轨迹状态，预测当前时刻目标可能出现的位置和尺度，并在新观测到来后进行修正，从而形成“预测—更新”闭环。该过程能够有效解决以下问题：

1. 目标尺度变化引起的匹配不稳定；
2. 短时漏检造成的轨迹断裂；
3. 检测噪声导致的轨迹抖动。

本文采用状态向量表示单条轨迹的运动状态：
\[
x = [u, v, \gamma, h, \dot{u}, \dot{v}, \dot{\gamma}, \dot{h}]^T
\]
其中 \((u,v)\) 为检测框中心坐标，\(\gamma = w/h\) 为宽高比，\(h\) 为框高，\((\dot{u},\dot{v},\dot{\gamma},\dot{h})\) 为对应状态的速度分量。

卡尔曼滤波包括预测与更新两个阶段：

#### 4.3.1 预测阶段
根据上一时刻状态 \(x_{t-1}\) 和状态转移矩阵 \(F\)，预测当前时刻状态：
\[
\hat{x}_t = Fx_{t-1}
\]
\[
\hat{P}_t = FP_{t-1}F^T + Q
\]

其中 \(Q\) 为过程噪声协方差矩阵，用于描述目标运动的不确定性。预测阶段的作用是为后续数据关联提供时序先验，使目标匹配不再完全依赖局部重叠信息。

#### 4.3.2 更新阶段
当当前帧检测结果到达时，将观测量 \(z_t\) 与预测状态进行融合，得到后验估计：
\[
K_t = \hat{P}_tH^T(H\hat{P}_tH^T + R)^{-1}
\]
\[
x_t = \hat{x}_t + K_t(z_t - H\hat{x}_t)
\]
\[
P_t = (I - K_tH)\hat{P}_t
\]

其中 \(K_t\) 为卡尔曼增益，\(H\) 为观测矩阵，\(R\) 为观测噪声协方差矩阵。更新阶段能够修正预测偏差，平滑目标轨迹，并抑制检测噪声引起的位置与尺度抖动。

通过卡尔曼滤波，本文实现了对目标下一时刻位置与尺度的预测，从而增强了轨迹的时间连续性，降低了短时漏检导致的 ID 丢失和轨迹终止风险。

### 4.4 数据关联与轨迹维护
在完成检测预处理和运动状态估计后，本文进一步通过数据关联与轨迹维护模块实现检测框与轨迹的稳定匹配。该部分主要解决 ID 切换、错误匹配和轨迹碎片化问题。

#### 4.4.1 数据关联机制
本文采用 IOU 与马氏距离联合构造关联代价矩阵，并利用匈牙利算法完成全局最优匹配。

对于检测框 \(z_i\) 与轨迹 \( \tau_j \)，其关联代价定义为：
\[
C_{ij} = \lambda \cdot d_{mahalanobis}(z_i, \tau_j) + (1-\lambda)\cdot(1-IOU(z_i,\tau_j))
\]

其中，\(d_{mahalanobis}\) 用于衡量观测与轨迹预测状态之间的运动一致性，IOU 用于衡量空间重叠程度。两者结合可以同时利用“运动趋势”和“空间重叠”两类信息，从而提高复杂场景下的匹配准确性。由于本文研究的场景中外观特征较为简化，因此关联过程更强调运动信息的作用。

在求解阶段，采用匈牙利算法对代价矩阵进行全局最优分配，使检测框与轨迹之间形成一一对应关系。相比贪心匹配，匈牙利算法能够避免局部最优导致的错误分配，更适合低空多目标跟踪中目标密集、交叉和遮挡频繁的情况。

#### 4.4.2 轨迹确认机制
为避免短期误匹配导致轨迹过早确认，本文对新轨迹采用待定状态管理。即新检测框首次匹配后不立即确认为稳定轨迹，而是要求其在连续若干帧内保持匹配成功，才能升级为确认轨迹。该机制能够有效抑制误检框造成的虚假轨迹，提高轨迹的可信度。

#### 4.4.3 轨迹删除机制
当轨迹在连续若干帧内无法与检测结果匹配时，说明该目标可能已经离开场景或确实失配。此时将其删除，以避免无效轨迹长时间保留并干扰后续关联。删除机制与卡尔曼滤波的短时保留机制配合使用，既能保留短时漏检下的目标连续性，又能及时清理长期失配的轨迹。

#### 4.4.4 轨迹维护效果
通过“确认—保留—删除”的轨迹生命周期管理，本文在保证实��性的同时，增强了轨迹的稳定性和可持续性。该机制能够有效缓解目标短暂消失、检测抖动和误检干扰导致的轨迹中断问题，从而显著减少 ID 切换和轨迹碎片化。

### 4.5 方法创新点分析
本文提出的低空多目标跟踪滤波方法的创新性主要体现在以下三个方面：

1. **面向城市场景仿真序列的检测结果预处理**  
   将 YOLOv8 检测结果筛选、置信度过滤和尺度归一化结合起来，提升输入序列稳定性，增强后续关联可用性。

2. **融合运动预测的状态估计机制**  
   利用卡尔曼滤波对目标位置和尺度进行时序预测，解决短时漏检、尺度变化与轨迹中断问题，提高轨迹连续性。

3. **联合关联与生命周期维护机制**  
   结合 IOU、马氏距离、匈牙利算法以及轨迹确认/删除机制，在复杂场景下降低误匹配和 ID 切换概率，提升轨迹管理能力。

### 4.6 本章小结
本章围绕城市场景仿真序列下的低空多目标跟踪任务，提出了一种由检测序列预处理、运动状态估计和数据关联与轨迹维护组成的低空多目标跟踪滤波方法。该方法针对目标尺度变化、检测噪声、短时漏检和 ID 切换等问题进行了针对性设计，其中检测预处理用于提升输入稳定性，卡尔曼滤波用于增强运动预测能力，数据关联与轨迹维护用于提高匹配准确性和轨迹连续性。实验结果表明，该方法能够有效提升复杂城市场景下低空多目标跟踪的稳定性，并减少 ID 切换。

## 5 仿真结果及分析
### 5.1 实验设置与数据说明
为验证本文构建的城市场景无人机检测序列生成模型的真实性与挑战性，以及跟踪方法在复杂观测条件下的鲁棒性，本文基于400帧标准化仿真检测序列开展定量统计与对比实验。实验所用检测序列由uav_simulator.m生成，包含透视投影畸变、高斯位置噪声、尺度噪声、随机漏检与泊松误检，能够较完整地模拟城市场景低空多目标跟踪的典型挑战。

### 5.2 检测序列统计特性与挑战性分析
本节从时序统计分布、检测置信度、目标尺度变化、观测抖动强度四个维度，对仿真检测序列进行全面量化分析，揭示跟踪难点的来源与机理。

### 5.3 跟踪输出稳定性对比分析
本节基于具有挑战性的检测序列，对IOU-greedy基线与引入卡尔曼滤波的DeepSORT跟踪方法进行量化对比，从活跃轨迹数、累计ID增长和轨迹长度分布等指标验证运动预测的核心作用。

### 5.4 结果成因分析
两种方法的性能差异本质源于运动预测的有无。IOU-greedy仅依赖帧间IOU匹配，无运动预测与时序先验，在透视投影引发的尺度变化、高斯噪声扰动和随机漏检共同作用下，相邻帧同一目标的IOU急剧下降，匹配频繁失败，最终导致轨迹碎片化与ID爆炸。DeepSORT则通过卡尔曼滤波提供的运动预测为数据关联提供时序先验，使匹配不再完全依赖局部IOU，并结合轨迹生命周期管理机制进一步抑制轨迹断裂和ID重生问题。

综上，卡尔曼滤波（运动预测）在反无人机集群仿真场景中，对降低轨迹碎片化、提升多目标跟踪关联稳定性具有不可替代的关键作用。

## 6 结论
本文面向城市场景仿真序列下的低空多目标跟踪任务，围绕尺度变化、检测噪声、短时漏检与ID切换等问题，提出一种融合检测序列预处理、卡尔曼滤波预测和多度量关联的低空多目标跟踪滤波方法。实验表明，该方法能够有效提升轨迹连续性，减少 ID 切换，并增强复杂城市场景下多目标跟踪的稳定性与鲁棒性。后续工作可进一步引入更丰富的外观特征与更复杂的三维运动模型，以提升在更强遮挡和更大规模目标场景中的适用性。
