**1 引 言**

随着低空领域开放与无人机技术的快速发展，由多架小型无人机构成的集群应用日益广泛。在带来便利的同时，无人机集群
"低、慢、小、数量多、易突防"
的特点，对重要区域安防、空域管控和反制无人机集群任务构成了严峻挑战。在实际对抗场景中，无人机常以编队形式密集飞行，目标之间易发生重叠、遮挡，且在透视投影下尺度变化剧烈，再加上检测噪声、漏检、误检等因素的影响，传统仅依靠帧间匹配的跟踪方法极易出现轨迹碎片化、ID
频繁切换、跟踪中断等问题，难以满足连续、稳定、可靠的目标跟踪需求。

目标检测与多目标跟踪是反无人机集群感知系统的核心环节，其性能直接决定后续态势分析、意图识别与拦截处置的效果。近年来，以
YOLOv8
为代表的深度学习检测模型，凭借高精度与高实时性优势，成为无人机目标检测的主流方案；以
DeepSORT
为代表的多目标跟踪算法，通过数据关联与轨迹管理实现连续跟踪；而卡尔曼滤波作为运动预测与状态估计模块，能够有效平滑噪声、预测短时漏检位置、提升关联稳定性，是改善跟踪性能的关键技术。与此同时，图像处理作为前端预处理环节，可抑制图像噪声、增强目标特征，进一步提升检测与跟踪的鲁棒性。

在典型反无人机系统架构中，多目标跟踪位于检测与告警之间，是连接单次观测与连续态势的核心模块。其主要作用包括从含噪声、漏检的检测序列中恢复目标连续轨迹；维持稳定ID，为目标计数、轨迹预测、行为分析提供支撑；抑制虚警、过滤抖动，提升系统输出可信度；为后续拦截、处置、驱离提供稳定目标状态。可以说，跟踪关联稳定性直接决定整个反无人机系统的可靠性。

由于真实无人机集群视频数据获取困难、场景不可控、标注成本高，难以开展可复现、可对比的量化实验。因此，本文依托仿真平台构建标准化检测序列，设计消融实验对比无预测跟踪方法与
YOLOv8+DeepSORT
组合的性能差异，重点验证卡尔曼滤波在提升跟踪连续性与关联稳定性方面的作用。通过系统研究与实验分析，为反无人机集群工程系统中的检测跟踪模块设计、算法选型与参数优化提供理论依据和实验支撑。

# 4 多目标跟踪方法与实现

现在围绕反无人机集群场景下的多目标跟踪任务，系统阐述从检测序列到连续轨迹的完整技术链路。首先形式化定义多目标跟踪问题，随后分别介绍无运动预测的
IOU-greedy 基线方法与融合卡尔曼滤波的 DeepSORT
跟踪框架，重点分析卡尔曼运动预测在提升关联稳定性中的核心作用，为后续实验验证奠定理论基础。

## 4.1多目标跟踪问题形式化定义

多目标跟踪（Multi-Object
Tracking,MOT）的核心任务是：从逐帧检测框序列中，为每个目标分配唯一且稳定的身份标识（ID），并输出连续的时空轨迹。

设仿真序列总帧数为$T$，第t帧（t=1,2,...,T）的检测结果为$D_{t} = {\{ z_{i}^{t}\}}_{i = 1}^{M_{t}}$，其中$M_{t}$为第t帧检测框数量，单个检测框$z_{i}^{t} = {\lbrack x_{i}^{t}，y_{i}^{t}，w_{i}^{t}，h_{i}^{t}\rbrack}^{T}$表示第i个目标的左上角坐标与宽高（像素）。

多目标跟踪的输出为轨迹集合
$\mathcal{T =}{\{\tau_{j}\}}_{j = 1}^{N_{track}}$

其中单条轨迹

$$\tau_{j} = \{(z_{j，t}，t)|t \in \lbrack t_{j，start}，t_{j，end}\rbrack\}$$

$N_{track}$为生成的独立轨迹ID总数。跟踪的核心目标是：在检测噪声、尺度变化、随机漏检等干扰下，实现检测框与轨迹的最优数据关联，最小化轨迹碎片化与
ID 切换，最大化轨迹连续性与 ID 保持能力。

## 4.2基线方法：IOU-greedy 无预测关联算法

### 4.2.1算法原理

IOU-greedy是典型的无运动预测跟踪方法，其核心逻辑为仅依赖帧间检测框的交并比（Intersection
over Union,
IOU）进行贪心匹配，无卡尔曼滤波预测、无轨迹状态管理、无时序先验约束。

对于第t帧的每个检测框$z_{i}^{t}$，计算其与第$t - 1$帧所有轨迹末端检测框$z_{j}^{t - 1}$的IOU：

$$IOU\left. （z_{i}^{t}，z_{j}^{t - 1} \right.） = \frac{Area\left. （z_{i}^{t} \cap z_{j}^{t - 1} \right.）}{Area（z_{i}^{t} \cap z_{j}^{t - 1}）}$$

选取IOU最大且超过预设阈值的轨迹进行匹配；若无满足条件的轨迹，则为该检测框新建轨迹ID。该方法逻辑简单、计算高效，但完全依赖局部空间重叠，对尺度变化、噪声扰动鲁棒性极差。

### 4.2.2算法局限性分析

在反无人机集群场景中，IOU-greedy的局限性被显著放大：

1.变化导致 IOU
失效：由3.2.3节透视投影关系w,h∝1/Z可知，目标深度变化会引发检测框尺度剧烈波动，相邻帧同一目标的IOU急剧下降，匹配失败概率大幅提升；

2.扰动加剧匹配歧义：高斯位置噪声与尺度噪声会导致检测框位置、尺寸抖动，进一步降低帧间IOU，引发误匹配；

3.直接导致轨迹断裂：无运动预测时，短时漏检会使轨迹失去匹配源，直接终止轨迹，目标重出现时触发新ID生成，最终导致严重轨迹碎片化。

## 4.3DeepSORT跟踪框架与卡尔曼滤波核心作用

DeepSORT（Simple Online and Realtime Tracking with a Deep Association
Metric）是基于检测的跟踪（Tracking-by-Detection）框架，通过卡尔曼滤波运动预测、匈牙利算法数据关联、轨迹生命周期管理三大核心模块，解决无预测方法的轨迹碎片化问题。本文重点突出卡尔曼滤波的运动预测作用，外观特征仅作为理论补充，不做强依赖。

### 4.3.1轨迹状态表示与卡尔曼滤波

DeepSORT采用8维状态向量描述单条轨迹的运动状态：

$$x = {\lbrack u,v,\gamma ，h，\dot{u}，\dot{v}，\dot{\gamma}，\dot{h}\rbrack}^{T}$$

其中(u,v)为检测框中心坐标，γ=w/h为宽高比，h为框高，$(\dot{u}，\dot{v}，\dot{\gamma}，\dot{h})$为对应状态的速度分量。

卡尔曼滤波分为预测与更新两个核心步骤：

1.预测步骤（Predict）：基于上一时刻状态$x_{t - 1\ }$与状态转移矩阵$F$，预测当前时刻的状态$\widehat{x_{t}}$与协方差$\widehat{P_{t}}$：

> $$\widehat{x_{t}} = Fx_{t - 1}$$
>
> $$\widehat{P_{t}} = FP_{t - 1}F^{T} + Q$$

其中$Q$为过程噪声协方差矩阵，用于描述运动模型的不确定性。预测步骤为数据关联提供时序先验，使匹配不再完全依赖局部
IOU。

2.更新步骤（Update）：融合当前帧检测观测$z_{t}$，对预测状态进行修正，得到最优后验估计$x_{t}$

与协方差$P_{t}$：

$$K_{t} = \widehat{P_{t}}H^{T}\left. （H\widehat{P_{t}}H^{T} + R \right.）^{- 1}$$

$$x_{t} = \widehat{x_{t}} + K_{t}\left. （z_{t} - H\widehat{x_{t}} \right.）$$

$$P_{t} = \left. （I - K_{t}H \right.）\widehat{P_{t}}$$

其中$K_{t}$为卡尔曼增益，$H$为观测矩阵，R为观测噪声协方差矩阵。更新步骤实现观测与预测的融合，平滑轨迹抖动，抑制噪声影响。

### 4.3.2数据关联机制

DeepSORT 采用运动信息与 IOU 联合关联策略，构建关联代价矩阵：

> $$C_{ij} = \lambda \bullet d_{mahalanobis}\left. （z_{i}，\tau_{j} \right.） + \left. （1 - \lambda \right.） \bullet \left. （1 - IOU\left. （z_{i}，\tau_{j} \right.） \right.）$$

其中$d_{mahalanobis}$
为马氏距离，用于衡量检测框与轨迹预测状态的运动一致性；λ为权重系数，本文中因仿真数据外观简化，λ取1，完全依赖运动信息关联。

基于代价矩阵，采用匈牙利算法求解最优匹配，实现检测框与轨迹的一一对应；同时通过马氏距离门限过滤异常匹配，降低ID切换概率。

### 4.3.3轨迹生命周期管理

DeepSORT 通过轨迹状态管理机制，进一步提升跟踪稳定性：

1.轨迹初始化：新检测框首次匹配时，创建tentative（待定）轨迹；

2.轨迹确认（confirmed）：若待定轨迹连续多帧成功匹配，则升级为confirmed轨迹，视为有效稳定轨迹；

3.轨迹保留（max_age）：当轨迹短时漏检时，通过卡尔曼滤波持续预测状态，保留轨迹max_age帧，避免漏检导致的轨迹断裂；

4.轨迹删除：若轨迹超过max_age帧无匹配，则删除该轨迹。

该机制有效抑制了漏检引发的轨迹重生，大幅提升轨迹连续性。

### 4.3.4卡尔曼滤波在跟踪中的核心贡献

卡尔曼滤波是DeepSORT提升关联稳定性的核心模块，其作用可总结为：

1.时序先验提供匹配依据：预测步骤为数据关联提供目标位置的时序预测，使匹配不再完全依赖局部
IOU，解决尺度变化、噪声扰动导致的IOU失效问题；

2.短时漏检补偿轨迹连续性：在无观测时持续预测目标位置，保留轨迹状态，避免漏检引发的轨迹断裂与ID重生；

3.噪声平滑抑制轨迹抖动：融合观测与预测，平滑检测噪声导致的框位置、尺寸抖动，提升轨迹输出的稳定性；

4.门限过滤降低误匹配：通过马氏距离门限过滤异常检测，减少误匹配与ID切换，提升关联准确性。

# 5 仿真结果及分析

为验证本文所构建的城市场景无人机检测序列生成模型及其对多目标跟踪的挑战性，本节从检测序列统计特性、尺度变化规律、噪声扰动强度以及跟踪输出稳定性四个方面进行分析。所有统计结果均由仿真输出的标准化检测CSV与跟踪结果文件自动生成，保证了分析过程的可复现性。

## 5.1实验设置与数据说明

为验证本文构建的城市场景无人机检测序列生成模型的真实性与挑战性，以及跟踪算法在复杂观测条件下的鲁棒性，本章基于
400
帧标准化仿真检测序列开展定量统计与对比实验。实验所用检测序列由uav_simulator.m生成，包含透视投影畸变、高斯位置噪声、尺度噪声、随机漏检与泊松误检，完全复现低空反无人机场景的典型不确定性。所有统计指标与曲线均由enhanced_trajectory_analysis.m与generate_analysis_report.m自动计算生成，确保实验可复现、可对比、可量化。

## 5.4 结果成因分析

两种方法的性能差异本质源于运动预测的有无：

IOU-greedy 失效根源：仅依赖帧间 IOU
匹配，无运动预测与时序先验，在透视投影引发的尺度变化、高斯噪声扰动、随机漏检的共同作用下，相邻帧同一目标的
IOU 急剧下降，匹配频繁失败，最终导致轨迹碎片化与 ID 爆炸；

DeepSORT
提升核心：卡尔曼滤波提供的运动预测为数据关联提供时序先验，使匹配不再完全依赖局部
IOU，在尺度变化、噪声、漏检条件下仍能维持轨迹连续；同时轨迹生命周期管理机制（max_age、confirmed）进一步抑制了漏检引发的轨迹断裂，从根本上解决了
ID 重生问题。

综上，卡尔曼滤波（运动预测）在反无人机集群仿真场景中，对降低轨迹碎片化、提升多目标跟踪关联稳定性具有不可替代的关键作用。

**5 结 论**

本文针对常规火焰面方程中组分扩散系数相同及刘易斯数为1的基本假设不适于层流非预混燃烧问题，根据所新定义的混合分数定义，详细推导得出了适用于层流非预混燃烧的相容性火焰面方程。然后通过对包含9组分28步反应的氢氧对撞层流非预混火焰进行数值仿真，证实了该火焰面方程的合理性，并得出了下面主要结论：

\(1\)
指出并改进了常用于求解微分方程稳态解的TWONPT软件包在计算节点更新及插值方式、雅可比矩阵更新、线性方程求解三个方面存在的不足，有效地提高该火焰面方程稳态解的计算效率和计算稳定性。

\(2\)
常规火焰面模型在火焰峰附近的温度分布，及OH、H~2~O、H~2~三种非单位刘易斯数组分的分布存在较大误差，而本文的相容性火焰面模型则能生成合理的火焰面库。

\(3\)
只有在氧化剂与燃气密度比及当量耗散率较大的条件下，变密度标量耗散率模型才会对仿真结果产生影响。
